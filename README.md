# SDE TEST Script

This is the sde test script

# Setup

## 1. Create a Virtual Environment

### On Unix or MacOS
```bash
python3 -m venv venv
```
### On Windows
```bash
python -m venv venv
```

## 2. Activate the Virtual Environment

### On Unix or MacOS

```bash
source venv/bin/activate
```

### On Windows
```bash
venv\Scripts\activate
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```
## 6. Run the Script
```bash
python main.py
```
